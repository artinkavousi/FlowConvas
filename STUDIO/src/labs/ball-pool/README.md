# Ball Pool Lab

Mode B conversion of mrdoob Ball Pool #2:

- Source: https://codepen.io/mrdoob/pen/dPpJMXB
- Preserved composition: WebGPU SSGI/TRAA/Bloom room, adaptive open-front box, Bounce rigid sphere pool, pointer-driven glass sphere/light, hold-to-respawn interaction.
- Canonical modules used: `webgpu-ssgi-room-renderer`, `adaptive-open-front-box-room`, `universal-physics-particles`, `bounce-rigid-sphere-adapter`, `pointer-glass-collider`.

The `modules/` folder is a snapshot of the canonical modules so this Lab remains copy-pasteable.
